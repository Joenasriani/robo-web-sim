# Quick Start Guide

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

```bash
git clone <repo-url>
cd robo-web-sim
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. Click **Launch Simulator** on the home page
2. Use the arrow keys or on-screen buttons to move the robot
3. Add commands to the queue and click **Run Queue** for automated sequences
4. Check the **Lessons** page for guided tutorials
